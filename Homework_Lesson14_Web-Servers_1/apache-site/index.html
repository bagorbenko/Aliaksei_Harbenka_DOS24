О JMeter

Apache JMeter — инструмент с открытым исходным кодом для нагрузочного тестирования и измерения производительности различных сервисов, включая веб-приложения.

Особенности

  • Тестирование производительности веб-сайтов, баз данных и FTP-серверов.
  • Поддержка различных протоколов: HTTP, HTTPS, SOAP, JDBC и других.
  • Гибкая архитектура и расширяемость через плагины.
